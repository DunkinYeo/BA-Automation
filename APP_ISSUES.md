## 이슈 목록

### APP-01 | 잘못된 시리얼 입력 시 에러 메시지
- **상태**: 🔍 확인 필요
- **재현 방법**: 로그인 화면에서 존재하지 않는 시리얼 번호(예: 000000) 입력 → 연결하기 클릭
- **현상**: "디바이스를 찾을 수 없음" 또는 "검사 정보 없음" 텍스트가 아닌 **숫자 에러 코드(902)** 포함 팝업이 표시됨
- **예상 동작**: TC 기준으로는 "검사 정보 없음" (ERR-03) 또는 "디바이스를 찾을 수 없음" (ERR-08/09) 텍스트 팝업 예상
- **확인 요청**: 서버 응답 코드 902의 의미 및 실제 표시되어야 할 에러 메시지 텍스트 확인

---

### APP-02 | GPS 꺼짐 에러 팝업 텍스트
- **상태**: 🔍 확인 필요
- **재현 방법**: 스마트폰 위치(GPS) 기능 OFF → 시리얼 입력 후 연결하기 클릭
- **현상**: 에러 팝업이 뜨나 "위치 기능이 꺼져 있습니다" 또는 "위치를 활성화해주세요" 등의 예상 텍스트가 감지되지 않음
- **확인 요청**: 해당 에러 팝업의 정확한 한국어/영어 텍스트 확인

---

### APP-03 | 블루투스 꺼짐 에러 팝업 텍스트
- **상태**: 🔍 확인 필요
- **재현 방법**: 스마트폰 블루투스 기능 OFF → 시리얼 입력 후 연결하기 클릭
- **현상**: APP-02와 동일하게 팝업 텍스트 감지 실패
- **확인 요청**: 해당 에러 팝업의 정확한 한국어/영어 텍스트 확인

---

### APP-04 | WiFi/네트워크 꺼짐 에러 팝업 텍스트
- **상태**: 🔍 확인 필요
- **재현 방법**: WiFi/모바일 데이터 OFF → 시리얼 입력 후 연결하기 클릭
- **현상**: APP-02와 동일하게 팝업 텍스트 감지 실패
