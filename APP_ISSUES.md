### APP-01
**잘못된 시리얼 입력 시 에러 메시지가 사용자 비친화적**

| 항목 | 내용 |
|------|------|
| 심각도 | 🔴 높음 |
| 화면 | 로그인 |
| 상태 | 개발 확인 필요 |

**재현 방법**
1. 로그인 화면에서 존재하지 않는 시리얼 번호 입력 (예: `000000`)
2. "연결하기" 버튼 클릭

**실제 동작**
```
팝업 제목 : 알 수 없는 오류 (902)
팝업 본문 : 알 수 없는 오류가 발생했습니다.
버튼      : 확인
```

**기대 동작**
사용자가 원인을 이해할 수 있는 메시지 표시  
예: "등록되지 않은 기기입니다." / "기기를 찾을 수 없습니다."

**확인 요청**
- 서버 응답 코드 `902`의 정의
- 해당 케이스에서 사용자에게 표시해야 할 공식 에러 메시지 텍스트

### APP-06
**GPS(위치) OFF 상태에서 BLE 연결 성공 및 에러 팝업 미표시**

| 항목 | 내용 |
|------|------|
| 심각도 | 🔴 높음 |
| 화면 | 로그인 → 연결, 검사 진행 중 |
| 상태 | 개발 확인 필요 |

**재현 방법 (케이스 1 — 연결 단계)**
1. 스마트폰 위치(GPS) 기능 OFF
2. 로그인 화면에서 시리얼 입력 후 "연결하기" 클릭

**재현 방법 (케이스 2 — 검사 진행 중)**
1. 검사 화면에서 검사 진행 중 (심박수 측정 중)
2. 스마트폰 위치(GPS) 기능 OFF

**실제 동작**
- 케이스 1: GPS OFF임에도 BLE 연결이 성공하여 "검사 정보 확인" 화면으로 진입 (에러 팝업 없음)
- 케이스 2: GPS OFF 후 20초 이상 경과해도 위치 꺼짐 에러 팝업 미표시, 검사 계속 진행

**기대 동작**
- 케이스 1: 연결 시도 시 위치 기능 꺼짐 에러 팝업 표시 후 연결 중단
- 케이스 2: GPS OFF 감지 후 일정 시간 내 위치 꺼짐 에러 팝업 표시

**스크린샷**
- 케이스 1: `output/20260514_150851/screenshots/20260514_150959_fail_test_login_err_11_gps_off.png`  
  → 상단에 "S-Patch AccurKardia · Foreground Service is running" 알림 확인 — BLE 실제 연결됨
- 케이스 2: `output/20260514_150851/screenshots/20260514_151257_fail_test_exam_err02_gps_off.png`  
  → 156 bpm 측정 중인 화면, GPS OFF 후에도 팝업 없이 계속 측정 중

**확인 요청**
- 연결 단계 및 검사 중 GPS 상태 체크 여부 및 체크 시점
- GPS 에러 팝업 표시 조건 및 정확한 텍스트 (한/영)

### APP-07
**BLE 연결 후 Back 복귀 시 재연결 실패 — "알 수 없는 오류 (302)"**

| 항목 | 내용 |
|------|------|
| 심각도 | 🔴 높음 |
| 화면 | 로그인 |
| 상태 | 개발 확인 필요 |

**재현 방법**
1. 시리얼 입력 후 "연결하기" → BLE 연결 성공 → "검사 정보 확인" 화면 진입
2. Back 버튼으로 로그인 화면 복귀
3. 같은 시리얼로 즉시 재연결 시도

**실제 동작**
```
팝업 제목 : 알 수 없는 오류 (302)
팝업 본문 : 알 수 없는 오류가 발생했습니다.
버튼      : 확인
```
재연결 불가 — 약 5초 후 재시도 시 정상 연결됨

**기대 동작**
Back으로 로그인 복귀 시 서버 세션이 정리되어 즉시 재연결 가능

**확인 요청**
- 서버 응답 코드 `302`의 정의
- Back 버튼으로 로그인 화면 복귀 시 서버 세션 정리(cleanup) 처리 여부
- 세션 정리까지의 예상 소요 시간

### APP-02
**GPS 꺼짐 에러 팝업 정확한 텍스트 미확인**

| 항목 | 내용 |
|------|------|
| 심각도 | 🟡 중간 |
| 화면 | 로그인 |
| 상태 | 텍스트 확인 필요 (APP-06 해결 후 재테스트 예정) |

**현황**  
APP-06 이슈로 인해 GPS OFF 시 에러 팝업 자체가 표시되지 않아 텍스트 확인 불가  
APP-06 수정 후 팝업이 정상 표시되면 자동화 TC에서 텍스트 캡처 예정

**확인 요청**  
GPS 꺼짐 에러 팝업의 정확한 텍스트 공유 (제목 / 본문 / 버튼, 한국어 기준)
